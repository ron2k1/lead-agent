# Code of Conduct

This project follows the Contributor Covenant 2.1.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness unacceptable behavior, contact the project
maintainer at ronilbasu@gmail.com. Reports will be reviewed within 7 days.

## Enforcement

The project maintainer reserves the right to remove, edit, or reject
comments, commits, code, issues, and other contributions that are not
aligned with this Code of Conduct, and will communicate reasons for
moderation decisions when appropriate.
